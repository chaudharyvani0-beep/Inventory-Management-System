# 📦 Inventory Management System

A simple console-based Inventory Management System built with Python. This project allows users to manage products using CRUD operations and stores data permanently using JSON file handling.

## ✨ Features

- Add Product
- View Products
- Search Product (by ID or Name)
- Update Product
- Delete Product
- Sell Product
- Low Stock Alert
- Restock Product
- Inventory Statistics
- JSON File Handling

## 🛠️ Technologies Used

- Python
- JSON
- File Handling

## 📂 Project Structure

```
main.py
inventory.py
helper.py
file_handler.py
inventory.json
```

## ▶️ How to Run

```bash
python main.py
```
